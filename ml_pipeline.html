Step 4 — GitHub Deployment Guide

Host your model on GitHub Pages for free — SahayCredit will fetch it in real time to score borrowers.

Step 01 · Create Repo
📁 Create GitHub Repository
  • Go to github.com/new
  • Name it: sahay-credit-model
  • Set visibility to Public
  • Check "Add a README file"
  • Click Create repository
🔗 Open GitHub
Step 02 · Upload File
⬆ Upload Your Model
  • Click "Add file" → "Upload files"
  • Drag & drop sahay_credit_model.json
  • Commit message: Add SahayCredit model v1
  • Click "Commit changes"
  • Wait for the upload to complete
Step 03 · Enable Hosting
🌐 Enable GitHub Pages
  • Go to your repo's Settings tab
  • Click Pages in the left sidebar
  • Source: Deploy from a branch
  • Branch: main → Folder: / (root)
  • Click Save
⏱ GitHub Pages takes 2–5 min to activate on first setup.
Step 04 · Get Your URL
🔗 Generate Model URL
  • Enter your GitHub username below
  • Copy the generated model URL
  • Paste it into your SahayCredit app
